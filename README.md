# Wati 
# 🚀 WATI Landing Page Clone

A pixel-perfect, responsive clone of the WATI landing page built using **HTML, CSS, and JavaScript**. This project focuses on clean UI implementation, modern design practices, and scalable code architecture using the **BEM (Block Element Modifier)** methodology.

---

## 🌐 Live Demo

🔗 https://wati-snowy.vercel.app/

---

## 📌 Features

* ✅ Pixel-perfect UI inspired by WATI
* ✅ Fully responsive (Desktop, Tablet, Mobile)
* ✅ Clean and scalable BEM class structure
* ✅ Modern layout using Flexbox & CSS Grid
* ✅ Reusable components (buttons, cards, sections)
* ✅ Smooth hover effects and UI interactions
* ✅ Mobile navigation toggle

---

## 🛠️ Tech Stack

* **CSS3**
